# Excel比对合并工具

## 功能说明
- 支持两个Excel文件比对，自定义匹配主键对齐数据
- 灵活选择要比对的字段和要输出的字段
- 差异自动标注：不一致内容标红，仅单表存在内容标黄
- 导出合并后的Excel文件

## 运行方法
### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 直接运行
```bash
python main.py
```

### 3. 打包成Windows EXE
运行打包脚本：
```bash
build.bat
```
打包完成后EXE文件在dist目录下。

## 使用步骤
1. 选择左右两个Excel文件，选择对应Sheet
2. 选择匹配主键（用来对齐两行数据，比如身份证号、订单号）
3. 选择要比对的字段（多选）
4. 选择要输出的字段（默认全选，可取消不需要的字段）
5. 点击开始比对，查看结果
6. 导出合并后的Excel文件
